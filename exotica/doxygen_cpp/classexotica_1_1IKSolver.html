Detailed Description

IK position solver The solver constructs and solves a linear program of the following form:

Where S is a diagonal matrix of task scaling factors rho for each task, jacobian is the Jacobian, W is a diagonal matrix of the joint space weights.

When regularisation term C is used, it gets appended to the Jacobian and distance to the nominal pose is appended to the y vector. For more details see: https://github.com/ipab-slmc/exotica/pull/465#issuecomment-449021817
